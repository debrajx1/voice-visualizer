# 🎧 Voice-to-Colorful Waveform Visualizer

A web app that visualizes your voice as a dynamic, colorful waveform using Web Audio API and `<canvas>`. Designed to be responsive and visually appealing.

## 🚀 Features

- Real-time microphone input
- Colorful animated waveform
- Theme switcher (rainbow, fire, pastel)
- Screenshot export
- Modular structure (easy to expand)

## 🛠️ Tech Stack

- HTML5 + CSS3
- JavaScript (ES Modules)
- Web Audio API
- Canvas API

## 📦 Folder Structure

Refer to `index.html` and modular `scripts/` for full separation of logic and visuals.
